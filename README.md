<h1 align="center">hero-cli</h1>

>  🌱  交互脚手架工具，项目脚手架多样，给同学们多一个选择。持续开发中。。。

<br />

## 特性
- 🚅: 可下载多类型模板脚手架，开发效率更高

### 主要插件
- commander: node.js命令行界面的完整解决方案
- inquirer: 规交互式命令行
- download-git-repo: git模板下载


### 安装

```shell
# install 
$ npm install -g hero-cli
# or yarn
$ yarn global add hero-cli
```

### 使用 

```shell
$ hero-cli create <name> [-f|--force]
```
